Numeris Data's App design

2021 • Webdesign & UX/UI

The interface design project for the Tereo application, distributed by Numéris Data, aimed to create an intuitive and efficient user experience for professionals.

Project Objectives:

Improve Ergonomics: Design a user interface that is simple to use, making navigation and access to various features easier.

Modernize Visual Appearance: Utilize a modern and attractive design to reflect Numéris Data's innovation and technical expertise, in line with their rebranding.

Optimize User Experience:Ensure a smooth and seamless user experience.

This project demonstrated our ability to create custom digital solutions focused on user needs while adhering to modern and innovative design standards.
